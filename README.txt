日本語専攻を強制的に決められた生徒は、そもそも興味がない...

■仕様
・ひらがな50音のみ（濁音、半濁音、拗音の文字はありません。）
・あかさたなのどの列を出題するかはランダムです。
・落下物の受け皿は、あいうえお順のみに対応（順番を覚えてもらうため）
・やゆよ、わをんの行も5文字になるようにしています。

■操作方法
・落ちてくる文字をマウスで掴んで正しいトレイに入れます。
・正解すると◎が表示されます。間違っていると✖が表示されます。
・✖でも、もう一度マウスで掴んで正しいトレイに入れることができます。
・全部正解するとかかった時間がポップアップします。

■課題
・テキストを問題に貼り付けできたら、色の変更など簡単にできると思うが、
　matter.jsでの方法が分からなかった。
　これができたら、カタカナ等の問題も早くできるかもしない。
　今回は力技でpngファイルを作成しました。
・PC用で作成したのでスマホでは盾が短い。
・得点計算などが入れられたら、生徒のやる気を刺激するかもしれない。

■フォルダ構成
mysite ---- f:index.html    : html本体
            f:style.css     : stylesheet
            f:main.js       : javascript
            f:matter.min.js : matterのengine
            f:good.png      : 正解の画像ファイル
            f:bad.png       : 不正解の画像ファイル
            d:char  ----  char00.png ~ char49.png : 問題用の画像ファイル
            d:tray  ----  tray00.png ~ tray49.png : 受け皿用の画像ファイル
※画像ファイルの数値（00～49）は正解判定で使用しているので変更不可
